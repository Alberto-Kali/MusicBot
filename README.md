# MusicBot Telegram WebApp

This branch contains only the Telegram WebApp source and its Docker build context.

## Image

CI publishes:
- `ghcr.io/alberto-kali/musicbot-telegram-webapp:latest`
- `ghcr.io/alberto-kali/musicbot-telegram-webapp:sha-<commit>`

Release changes for production are made separately in the `main` branch by updating `docker-compose.yml`.
